FoodieApp

1) FoodieApp is an application which enable users to register and login and can see all the restaurants.
2) Users can be able to see restaurant nearby their location.
3) They can also add thrie favourite restaurants in the favourite section of the application.
4) As well as users can keep their favourite cuisines in the cart.
5) Admin can add the restaurants as well as can edit the food items available in the restaurants.
6) Languages and tools used is IntelliJ IDE, VS-Code, Java, Angular, SpringBoot, MySql, MongoDB.
